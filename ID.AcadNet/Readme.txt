Lsds (v2.0.11.3)

1. Settings:
1.1 User ROUMING PATH: "C:\Users\<USER>\AppData\Roaming\Autodesk\AutoCAD 2013 - English\R19.0\enu\support", where <USER> is user's computer name
1.2 Lsds plugin ROOT PATH: "C:\Program Files\Autodesk\ApplicationPlugins"

2. Deployment:
2.1 delete (or save to zip) all files from two paths: ROUMING(1.1) and ROOT(1.2)
2.2 copy bundle.zip to Root path
2.3 !!! Optional commands only for many users work: NETLOAD -> ***.DLL -> ***REG (or ***UNREG)
2.4 restart AutoCAD
